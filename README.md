This is a memo application created for learning Django.

## Usage
```shell
git clone https://github.com/mitamam/my_memo_app.git
cd my_memo_app/

pip install -r requirements.txt

# Generate SECRET_KEY
./gen_key.sh

python manage.py migrate
python manage.py runserver
```
